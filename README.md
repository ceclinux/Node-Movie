#Node-Movie

一个用来给家里播放电影的服务器架构。通过web访问

##如何使用

```
git clone 这个repo
npm install
mkdir movies
```

然后把你的字幕啊，电影啊统统扔进去，高清啊，1080p啊，无码啊...

最后

```
gulp
```

打开你的另外随便一台能上网的机子，输入`服务器ip:3000`

我承认我是给我的`Raspberry Pi`用的，拖一个硬盘，呵呵
